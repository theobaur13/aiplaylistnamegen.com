- AI Playlist Name Generator -

What is this?

This is a Flask-based website that enables users to upload their Spotify playlists to generate a tailored playlist name using GPT-3.5. This website is hosted using Vercel.

How to use

  1. Click "Upload playlist"
  2. Connect your Spotify account
  3. Click the playlist that you want to generate a new name for
  4. If you want to generate a new name, click "Generate another playlist name"

How it works

Your Spotify account is linked to the website using Spotify's API. We don't store any data but the site needs access to:

  • -Your spotify account type
  • -Your email address
  • -Your playlists
  • -Your collaborative playlists

We read each song and artist name in your selected playlist and pass it into the text-davinci-003 model (part of the GPT 3.5 family) to generate a name specific to the songs in your playlist.
